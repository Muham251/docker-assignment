# Docker Assignment: FastAPI & Next.js

This repository contains two containerized applications as part of the Docker assignment.

## Projects
1. **FastAPI App**: A Python backend running on port 8000.
2. **Next.js App**: A React frontend running on port 3000.

## How to Run
From the respective directories, run:
`docker build -t <name> .`
`docker run -p <port>:<port> <name>`
